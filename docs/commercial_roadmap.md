# YouthMentor AI — Commercial Roadmap

Prototype pricing for product discussions. **Not live billing** — Settings includes a "Premium demo" toggle.

## Consumer (B2C)

| Tier | Price (SGD) | Messages / day | Features |
|------|-------------|----------------|----------|
| Free | $0 | 20 | 2 starter coaching packs |
| Premium | $4.99 / mo | Unlimited | All packs, optional parent add-on |
| Family | $7.99 / mo | Unlimited | Parent dashboard + quiet hours |

## School (B2B) — Insights

| Tier | Price (SGD) | Data policy |
|------|-------------|-------------|
| Pilot | Free (1 term) | Anonymous aggregates only — no chat text |
| School | $800 / year | Mood trends, scenario usage |
| District | Custom | Multi-school API export |

**Privacy:** School tier never receives individual chat transcripts (PDPA-minimisation).

## Revenue logic

> Free tier builds trust; premium unlocks depth; schools pay for aggregate insights without accessing private conversations.

## Cost assumptions

| Item | Estimate |
|------|----------|
| LLM API per active user / month | ~$0.15–0.50 |
| Vercel hosting | $0 (hobby) → $20/mo (pro) |
